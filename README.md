# Ajmal M - Minimal Showcase

A lightweight one-page showcase for research, selected work, and contact links.

## Files

- `index.html` - Page content and structure
- `style.css` - Visual design and responsive layout

## Preview

Open `index.html` directly in a browser. No build step or framework is required.

## Customize

Update the email, profile links, research highlights, and selected work directly in `index.html`.
