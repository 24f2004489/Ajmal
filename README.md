# Ajmal M - Professional Portfolio

A minimal, clean professional portfolio site showcasing AI research and projects.

## Features

- 🎨 **Minimal Design** - Clean, light aesthetic with dark mode
- 📱 **Fully Responsive** - Works perfectly on all devices
- ⚡ **Fast & Lightweight** - Pure HTML/CSS, no frameworks
- 🔍 **SEO Friendly** - Semantic HTML structure
- 🎯 **Easy to Customize** - Simple structure for quick updates

## Sections

- **About** - Brief bio and key information
- **Research & Experience** - Research positions and achievements
- **Publications** - Recent and submitted papers
- **Projects** - Technical projects and work samples
- **Achievements** - Awards and recognitions
- **Contact** - Ways to get in touch

## How to Deploy

This site is designed to be hosted on GitHub Pages:

1. Push all files to your repository
2. Go to Settings → Pages
3. Select "Deploy from a branch"
4. Choose `main` branch
5. Your site will be live at `https://24f2004489.github.io/Ajmal`

## Customization

### Update Contact Information
Edit the contact section in `index.html` with your actual links:
- Phone number
- LinkedIn profile
- GitHub profile
- Email address

### Change Colors
Edit the CSS variables at the top of `style.css`:
```css
:root {
    --primary-color: #0f172a;
    --accent-color: #3b82f6;
    /* ... etc */
}
```

### Add/Update Content
Simply edit the HTML sections in `index.html` to add or modify content.

## Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, CSS Variables)
- No JavaScript required
